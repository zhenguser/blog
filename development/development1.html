# Linux系统配置静态IP地址

  1. 获取ipd地址：
dhclient
1
  1. 查看ip地址：
ip addr
1
  1. 修改IP地址:
vi /etc/sysconfig/network-scripts/ifcfg-ens33
1
  1. 修改文件内容,将ONBOOT=no改为ONBOOT=yes，表示网卡随系统一起启动，将BOOTPROTO=dhcp改为BOOTPROTO=static，dhcp表示自动获取IP地址，static表示手动设置静态IP地址。
    增加这几行内容：
IPADDR=192.168.30.128
NETMASK=255.255.255.0
GATEWAY=192.168.30.1
DNS1=119.29.29.29
1
2
3
4
  1. 重启网络服务:
systemctl restart network.service
1

# Linux安装Docker

  1. 通过uname -r命令查看你当前的内核版本:
uname -r
1
  1. 使用 root 权限登录 Centos。确保 yum 包更新到最新:
yum -y update
1
  1. 卸载旧版本(如果安装过旧版本的话):
yum remove docker docker-common docker-selinux docker-engine
1
  1. 安装需要的软件包，yum-util提供yum-config-manager功能，另外两个是devicemapper驱动依赖:
yum install -y yum-utils device-mapper-persistent-data lvm2
1
  1. 添加阿里云的docker源
yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
1
  1. 可以查看所有仓库中所有docker版本，并选择特定版本安装:
yum list docker-ce --showduplicates | sort -r
1
  1. 安装docker:
yum install docker-ce-17.12.0.ce
1
  1. 启动并加入开机启动:
systemctl start docker
systemctl enable docker
1
2
  1. 验证安装是否成功(有client和service两部分表示docker安装启动都成功了):
docker version
1

# Linux安装Redis

  1. 上传Redis的安装包
redis-5.0.6.tar.gz
1
  1. 解压我们的Redis安装包
tar -zxvf redis-5.0.6.tar.gz
1
  1. 创建目录
mkdir /usr/redis'
1
  1. 编译redis
make install PREFIX=/usr/redis
1
  1. 将Redis设置为后台启动
cp /usr/redis-5.0.6/redis.conf    /usr/redis/bin
vi redis.conf  daemonize yes
./redis-server ./redis.conf 重启启动Redis
ps aux | grep 'redis'
1

  1. 启动Redis
cd /usr/redis/bin     
./redis-server
